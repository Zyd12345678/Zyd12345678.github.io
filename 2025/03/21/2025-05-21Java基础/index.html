为什么要配置环境变量？

配置完环境变量，可以在任何位置运行Java命令。

常用快捷键

快捷键 作用
Ctrl+D 复制当前行到下一行
Ctrl+Y 删除当前行
Ctrl+Alt+L 格式化代码
Alt+Shift+↑ 上移当前代码
Alt+Shift+↓ 上移当前代码
Ctrl+/ 注释
Ctrl+Shift+/ 注释

Idea中模块操作

修改模块

选择模块然后右键，Refactor -> Rename -> Rename module and directory

导入模块

  1. 复制模块然后粘贴到Project里（此时没有进行关联）
  2. File -> New -> Module from Existing Sources… 然后找到要导入的模块(一顿操作点击即可overwrite)
  3. 配置Jdk -> Setup Sdk
  4. 如果爆红就重启idea

注释

  1. 单行注释：// 用于方法内部
  2. 多行注释：/* */ 用于方法内部
  3. 文档注释：/** */ 多用于类和方法上

类型转换

浮点型转为整数型，去掉小数，直接转换。

表达式的自动类型提升

在表达式中，小范围类型的变量，会自动转换成表达式中较大范围的类型，再参与运算。

表达式的最终结果类型由表达式中的最高类型决定。
在表达式中，byte、short、char是直接转成int类型参与运算的。

+号在Java中的特殊用途

  • “+”有些时候可以做连接符
  • “+”与字符串运算的时候用作连接符，其结果依然是个字符串

如何识别”+”是做运算还是连接符？
能算则算，不能算就连接
