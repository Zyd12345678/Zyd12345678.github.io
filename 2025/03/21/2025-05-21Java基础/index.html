常用快捷键

Idea中模块操作

修改模块

选择模块然后右键，Refactor -> Rename -> Rename module and directory

导入模块

  1. 复制模块然后粘贴到Project里（此时没有进行关联）
  2. File -> New -> Module from Existing Sources… 然后找到要导入的模块(一顿操作点击即可overwrite)
  3. 配置Jdk -> Setup Sdk
  4. 如果爆红就重启idea

注释

  1. 单行注释：// 用于方法内部
  2. 多行注释：/* */ 用于方法内部
  3. 文档注释：/** */ 多用于类和方法上
